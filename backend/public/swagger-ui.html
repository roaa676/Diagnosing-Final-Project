منصة الكشف المبكر عن صعوبات التعلم
Chatbot API v1.0
خطوات التجربة:
  1. اضغط POST /login → Try it out → سجّل دخولك واحصل على الـ token
  2. اضغط Authorize 🔒 في الأعلى → الصق الـ token
  3. جرّب POST /chatbot/ask باختر example من القائمة وغيّر الـ message لأي سؤال عربي
  4. جرّب POST /chatbot/explain-result و recommend-exercises بنفس الطريقة
⚠️ تذكير: المساعد لا يُشخّص طبياً. كل الردود مؤشرات أولية مبنية على أداء الطفل داخل المنصة.
